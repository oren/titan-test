# Titan 0.9 in Docker

## Run Titan

```
git clone https://github.com/oren/titan.git
cd titan
docker-compose up
```

## Run Gremlin client

```
npm install
node get-all.js
Error: socket hang up
```

## Gremlin console

    bin/gremlin
